# Fuzzy Logic Control of a Ball-and-Beam System: Experimental Evaluation with PID Benchmarking​
This repository contains our project for 49275 Neural Networks and Fuzzy Logic at the University of Technology Sydney.

## Project Overview

The **ball-and-beam system** is a classic nonlinear and inherently unstable control problem. The objective is to control the position of a ball rolling along a beam by adjusting the beam’s angle.

While many existing studies compare **PID** and **fuzzy logic controllers**, a large portion of this research relies on simulation tools such as MATLAB and Simulink. Although simulations are useful for initial validation, they often fail to capture the complexities of real-world systems.

This project addresses that gap by implementing and evaluating both controllers on a **physical system**, enabling a more realistic comparison of their performance.

---

## Project Aim

The aim of this project is to:

- Design and build a **physical ball-and-beam system**
- Develop and implement:
  - A **PID controller**
  - A **fuzzy logic controller**
- Experimentally evaluate and compare controller performance using real-world data

---

## Methodology

1. Construct the physical ball-and-beam apparatus  
2. Analyse system dynamics  
3. Design PID and fuzzy logic controllers  
4. Implement controllers on an Arduino-based system  
5. Evaluate performance using standard metrics:
   - Settling time  
   - Overshoot  
   - Steady-state error  
6. Compare and analyse results  

---

## Project Timeline

The project has been structured into six key phases:

- **Planning & Research**  
  Literature review, defining objectives, and establishing project scope and timeline.

- **System Design**  
  Developing system architecture, selecting components, and designing the mechanical structure.

- **Physical Development**  
  Building the ball-and-beam apparatus, integrating hardware (servo, Arduino, sensors), and validating functionality.

- **Controller Implementation**  
  Designing, implementing, and testing both PID and fuzzy logic controllers on the physical system.

- **Evaluation & Testing**  
  Measuring system performance and comparing controllers using standard control metrics.

- **Documentation & Presentation**  
  Preparing seminars and compiling the final research report.
